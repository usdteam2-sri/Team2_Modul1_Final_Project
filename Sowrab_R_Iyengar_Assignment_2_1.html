Problem # 2.17.

An instructor gives a course grade of B to students who have total score on exams and homeworks between 800 and 900, where the maximum possible is 1000. If the total scores have approximately a normal distribution with mean 830 and standard deviation 50, about what proportion of the students receive a B?

Your answer goes here

In[]:
from scipy.stats import norm

# Parameters of the normal distribution
mean = 830
std_dev = 50

# Boundaries for the grade B
lower_bound = 800
upper_bound = 900

# Calculate the cumulative probability for the boundaries
probability_lower = norm.cdf(lower_bound, mean, std_dev)
probability_upper = norm.cdf(upper_bound, mean, std_dev)

# Proportion of students receiving a B
proportion_b = probability_upper - probability_lower

print(f"Proportion of students receiving a B: {proportion_b:.4f}")

import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import gamma

# Fixed shape parameter k
k = 3

# Scale parameters to vary
scale_params = [0.5, 1, 2, 3, 4, 5]

# Generate x values for plotting
x = np.linspace(0, 20, 1000)

# Plot Gamma distributions with different scale parameters
plt.figure(figsize=(12, 8))

for scale in scale_params:
    y = gamma.pdf(x, a=k, scale=scale)

Problem # 2.27.

The distribution of X = heights (cm) of women in the U.K. is approximately N(162, $7^2$). Conditional on X = x, suppose Y = weight (kg) has a N(3.0 + 0.40x, $8^2$) distribution. Simulate and plot 1000 observations from this approximate bivariate normal distribution. Approximate the marginal means and standard deviations for X and Y. Approximate and interpret the correlation.

Your answer goes here

In[]:
import numpy as np
import matplotlib.pyplot as plt

# Given parameters for X ~ N(162, 7^2)
mu_x = 162  # mean of X 
sigma_x = 7  # standard deviation of X

# Given parameters for Y | X ~ N(3.0 + 0.40*X, 8^2)
mu_y_given_x_intercept = 3.0
mu_y_given_x_slope = 0.40
sigma_y_given_x = 8  # standard deviation of Y given X

# Number of observations to simulate
n = 1000

# Simulate X from N(162, 7^2)
X = np.random.normal(mu_x, sigma_x, n)

# Simulate Y conditional on X from N(3.0 + 0.40*X, 8^2)
Y = np.random.normal(mu_y_given_x_intercept + mu_y_given_x_slope * X, sigma_y_given_x)

# Plotting the simulated data
plt.scatter(X, Y, alpha=0.5)
plt.title('Simulated Bivariate Normal Distribution of Height and Weight')
plt.xlabel('Height (cm)')
plt.ylabel('Weight (kg)')
plt.grid(True)
plt.show()

# Approximating marginal means and standard deviations
mean_x = np.mean(X)
std_x = np.std(X)
mean_y = np.mean(Y)
std_y = np.std(Y)

# Calculate the correlation between X and Y
correlation = np.corrcoef(X, Y)[0, 1]

mean_x, std_x, mean_y, std_y, correlation
